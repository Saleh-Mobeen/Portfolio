Scratch

I began my coding journey with Scratch, a visual programming language that made learning the basics of logic and programming both fun and interactive. It helped me understand the foundational concepts like loops, conditionals, and event-driven programming.

CSS

To make my websites more engaging, I learned CSS, which allowed me to design and style web pages creatively. I explored everything from basic styling, like colors and fonts, to advanced features like animations, transitions, and responsive design. Mastering CSS taught me how to make my projects visually appealing while maintaining a user-friendly experience across devices.

Python

Python stood out as a multipurpose language in my learning journey. I used it for automation tasks, testing, and creating small projects like speedrun timer. Its simplicity made it ideal for exploring new concepts quickly, while its versatility opened doors to advanced fields like machine learning and web development.

Node.js

Node.js was my gateway to full-stack development. I learned how to create RESTful APIs, handle real-time communication, and build local server systems. By leveraging JavaScript on both the client and server sides, I developed efficient and seamless web applications, bridging the gap between end-user's device and backend server development.

React Native

In addition to React, I plan to dive into React Native, which will allow me to build cross-platform mobile applications using a single codebase. With React Native, I aim to create apps that deliver a native-like experience on both Android and iOS. This aligns with my vision of broadening my development skills and creating versatile applications that can reach a wider audience.

The real coding

After mastering the basics of logic and programming with Scratch, I wanted to explore the world of real code. Scratch had given me a solid foundation, but I was eager to dive deeper into how software and websites are built. This led me to begin learning HTML and CSS, where I started to write actual code to create my own web pages.

HTML

My journey into web development started with HTML, the backbone of every website. I learned to structure web pages using semantic elements, ensuring clarity and accessibility. From simple headings and paragraphs to embedding multimedia and forms, HTML taught me how to present information effectively on the web.

Javascript

JavaScript was the game-changer that transformed my static websites into dynamic, interactive applications. I learned how to manipulate the DOM, handle events, and integrate APIs to create features like form validation, sliders, and interactive menus. My growing expertise in JavaScript set the foundation for more advanced front-end and full-stack development.

PHP

PHP introduced me to the world of server-side programming. With PHP, I built dynamic web pages that could process user inputs, interact with databases, and manage sessions. It strengthened my understanding of backend logic and the importance of security in web applications.

React

I aim to learn React Native to expand my skills in mobile app development. Its ability to create cross-platform apps using a single codebase makes it an ideal choice for building versatile and efficient applications. With React Native, I plan to develop apps that deliver a seamless, native-like experience on both Android and iOS, reaching a broader audience.
